# Hero Born FPS肉鸽生存射击游戏

## 项目介绍
基于Unity3D+C#开发的快节奏生存游戏，融合FPS射击与Roguelike肉鸽元素，参考《土豆兄弟》数值体系，实现"战斗收集-成长强化-挑战进阶"的完整游戏循环。

## 核心玩法
- 躲避敌人攻击，收集场景道具
- 利用武器消灭敌人，获得资源
- 自由加点升级，构建独特战斗流派
- 难度递进，挑战更高波次

## 技术栈
C#、Unity3D、有限状态机(FSM)、NavMesh、UGUI、XML/JSON

## 核心功能
- 模块化面向对象架构，拆分12个独立模块，低耦合高内聚
- 基于FSM开发敌人AI，支持巡逻、追击、自动射击与受击反击
- 设计泛型销毁基类统一管理临时对象，解决内存泄漏问题
- 实现XML/JSON双格式数据持久化，支持武器库存与关卡进度保存
- 第一人称/第三人称双视角无缝切换
- 完整UGUI界面（主菜单、战斗HUD、胜负结算）

## 运行方式
1. 打开Unity 2022及以上版本
2. 导入本项目
3. 打开`Assets/Scenes/MainScene`场景
4. 点击运行按钮即可体验
